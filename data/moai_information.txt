- anaconda_path.txt 에 anaconda3/envs 경로를 적으면 됨
- ex) C:\Users\twim\anaconda3\envs

## 2. 화면 및 기능 설명

1. 학습 모드 선택 화면 (PPT 4 페이지)
- 학습 모드를 선택할 수 있음 (Object detection, Segmentation, Classification 중 택 1)
- **MOAI START** 버튼을 클릭하여 선택한 학습 모드로 프로그램을 실행

1. 메인 화면
    1. Object detection (PPT 6 페이지)
        - **이미지 폴더추가** 버튼: 폴더를 선택하여 이미지를 리스트에 불러옴
        - **이미지 삭제** 버튼: 선택한 사진을 리스트에서 삭제
        - **이미지 폴더삭제** 버튼: 이미지 리스트를 전부 삭제
        - **Train** 버튼: 이미지 리스트 중 학습 데이터를 리스트에 표시함
        - **Validation** 버튼: 이미지 리스트 중 검증 데이터를 리스트에 표시함
        - **불량 타입 추가** 버튼: 입력한 문자열로 불량 마킹 타입을 추가 (**불량 마킹 타입은 영문 또는 숫자만 가능**)
        - **불량 타입 제거** 버튼: 불량 타입 리스트에서 선택한 불량 마킹 타입을 제거
        - **화살표** 버튼: 불량 타입 리스트에서 불량 마킹 타입의 순서를 변경
    2. Classification (PPT 8 페이지)
        - **이미지 폴더추가** 버튼: 폴더를 선택하여 이미지를 리스트에 불러옴
        - **이미지 폴더삭제** 버튼: 이미지 리스트를 전부 삭제
        - **클래스 추가** 버튼: 분류 타입을 추가
    3. Segmentation (PPT 10 페이지)
        - **이미지 폴더추가** 버튼: 폴더를 선택하여 이미지를 리스트에 불러옴
        - **이미지 폴더삭제** 버튼: 이미지 리스트를 전부 삭제
        - **이미지 삭제** 버튼: 선택한 사진을 리스트에서 삭제
        - **불량 타입 추가** 버튼: 입력한 문자열로 불량 마킹 타입을 추가 (**불량 마킹 타입은 영문 또는 숫자만 가능**)
        - **불량 타입 제거** 버튼: 불량 타입 리스트에서 선택한 불량 마킹 타입을 제거
        - **화살표** 버튼: 불량 타입 리스트에서 불량 마킹 타입의 순서를 변경

1. 학습 화면 (PPT 12 페이지)
- 선택된 학습 모드에 따라 mAP 또는 loss 그래프를 표시
    - Object detection : mAP
    - Segmentation, Classification : loss
- EPOCH : 학습이 몇 번 진행되었는지 나타냄
- mAP or loss : 현재 학습중인 모델의 평가 척도를 나타냄
    - mAP : 모델이 예측한 바운딩 박스와 실제 바운딩 박스가 얼마나 유사한지(겹치는지) 를 나타내며, 높을수록 좋음
    - loss :
- **모델 생성** 버튼: 현재 학습된 모델의 ONNX 를 생성
- **학습 중지** 버튼: 현재 진행중인 학습을 중지

1. 모델 테스트 화면 (PPT 15 페이지)
- **사진선택** 버튼 : 선택한 폴더 내 이미지를 이미지 리스트로 불러옴
- **사진삭제** 버튼 : 이미지 리스트를 전부 삭제
- **모델선택** 버튼 : 테스트를 진행할 모델을 선택
- **분석시작** 버튼 : 선택한 모델로 불러온 이미지들을 테스트
- **Score 미표시** 옵션 : 결과 화면에서 score 정보를 표시하지 않음
